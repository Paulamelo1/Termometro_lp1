# Atividade de Nivelamento (Termômetro)- LP1 🚀

Este repositório contém os exercícios de nivelamento em C desenvolvidos para a disciplina de **Linguagem de Programação 1**. O objetivo é demonstrar o desempenho inicial em lógica de programação e fundamentos da linguagem.

**Aluna:** Ana Paula Evangelista Melo Freitas.

---

## 📂 Exercícios Desenvolvidos
* `lateN.c`: Imprime números sequenciais de 1 até um valor N fornecido pelo usuário.
* `contarcarac.c`: Contador de caracteres em strings.
* `fatorial.c`: Cálculo de fatorial.
* `primouunao.c`: Verificador de números primos.
* `somaarray.c`: Soma de elementos em vetores.
* `temperatura.c`: Conversor Celsius/Fahrenheit.

---

## Como Rodar
Para evitar erros de permissão de sistema ou bloqueios de arquivos no Windows/Linux (como o erro EPERM ou Permission Denied), siga rigorosamente os passos abaixo via terminal:

Evite utilizar o botão de "Play" de extensões automáticas. No terminal integrado do VS Code (Ctrl + '), digite:

1. Compile: `gcc nome_do_arquivo.c -o programa.out`
2. Execute: `./programa.out`

## Observações de Desenvolvimento
Todos os códigos foram salvos com a extensão .c para garantir a compatibilidade com o compilador GCC.

Foram aplicados conceitos de boas práticas e comentários para facilitar a correção.
